Note: 
ZoomAudioDevice.driver is used to share computer audio. Before using the shared computer audio function, make sure that ZoomAudioDevice.driver is installed to the system path "/Library/Audio/Plug-Ins/HAL". This step needs to be implemented by the user in their own code.

Usage: 
Copy ZoomAudioDevice.driver to "/Library/Audio/Plug-Ins/HAL" folder path.
